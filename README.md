# 📊 Facebook Engagement Analysis

## 📌 Overview
This project analyzes Facebook post engagement using a dataset and demonstrates web scraping techniques. The goal is to understand which types of posts perform best and how engagement varies.

---

## 🚀 Features
- 🔹 Web Scraping using BeautifulSoup (demo)
- 🔹 Data Cleaning using Pandas
- 🔹 Engagement Calculation (likes + comments + shares)
- 🔹 Data Visualization using Matplotlib & Seaborn
- 🔹 Interactive Dashboard using Streamlit

---

## 🛠️ Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Streamlit
- BeautifulSoup
